# Komtek_test
ТЕСТОВОЕ ЗАДАНИЕ ООО "КОМТЕК"
Версия Delphi - 10.3
Для работы с базами данных используются компоненты FireDAC(стандартные для Delphi c версии 10.1);
Сервер баз данных - MySQL, т.к. oracle или pg разворачивать у себя не хотел, а тестовая бд. уже была. Все настройки подключения прописаны в компоненте - FDConnection1;
Для корректной работы драйвера подключения к базе данных при первом запуске указать путь к библиотеке libmysql.dll(лежит в корне проекта);
Если будете тестировать на своей БД, то достаточно перенастроить компонет FDConnection1.
SQL запросы использованные в проекте +- стандартны и должны работать на любой БД.
